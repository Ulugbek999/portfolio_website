Recent projects

Name Comparison Tool

A high-performance Python application designed for cross-database record validation. Initially built for name matching, this tool scales to compare any structured data across Excel sheets or CSV files. It utilizes the pandas library for fast, vectorized operations and supports fuzzy matching for robust comparison of fields that may contain minor variations or formatting inconsistencies.

The tool is capable of processing tens of thousands of rows in seconds, making it ideal for validating records from multiple data sources. A simple yet functional GUI was developed using additional Python libraries to allow non-technical users to run comparisons without writing code.

AI Chatbots

A Python-based conversational AI system built using the OpenAI API and deployed through a lightweight Flask web interface. The chatbot supports intelligent, multi-turn conversations with dynamic context retention, making it suitable for task-specific assistance.

The backend leverages SQLite for session and user data management, regex for input pattern recognition and preprocessing, and requests for API handling. Data is structured and processed efficiently using pandas, and dotenv is used for secure configuration management.

The frontend is implemented with clean HTML and CSS, providing a responsive interface with minimal use of JavaScript. This project demonstrates how to architect and deploy AI-driven tools with a focus on modular design, data persistence, and real-world usability.
